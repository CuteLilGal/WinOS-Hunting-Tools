# WinOS-Hunting-Tools
Executable/Script/Other for Hunting on Microsoft Windows OS

**Invoke-FileAudit**: A PowerShell script for auditing binaries (DLL, EXE, SYS, ...) on Windows. Collects cryptographic hash, Authenticode signature, version info, and file metadata into CSV / JSON. Built for enterprise software inventory, supply-chain auditing, and unsigned-binary hunting.
